## Setup

```bash
# install
yarn install

# start
yarn dev

# format code
yarn fix

# test
yarn test
```

## Stack

- [React](https://reactjs.org/)
- [MUI](https://mui.com/)
- [Highcharts](https://www.highcharts.com/) ([highcharts-react](https://github.com/highcharts/highcharts-react))
- [react-use-bitbank](https://github.com/daikiojm/react-use-bitbank)
